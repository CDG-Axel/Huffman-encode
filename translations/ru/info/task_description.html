Мерзкими числами зовутся числа, состоящие только из простых делителей 2, 3 или 5. Последовательность 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, ... показывает первые 11 мерзких чисел (1 тоже считается мерзким числом). Напишите программу, которая находит мерзкое число с номером n. Все как в прошлой задаче с парой отличий - ограничение для входного значения теперь 1000000, а выходное значение - строка.

Входные данные: N, int.

Выходные данные: Мерзкое число с номером N, str.

Примеры:

{% if is_js %} 
extremely_ugly(4) == '4'
extremely_ugly(6) == '6'
extremely_ugly(11) == '15'
extremely_ugly(999) == '51018336'
{% else %} 
extremely_ugly(4) == '4'
extremely_ugly(6) == '6'
extremely_ugly(11) == '15'
extremely_ugly(999) == '51018336'
{% endif %}

Где используется подобное: Для демонстрации того, что простое решение может быть весьма медленным

Ограничения: Входное число меньше или равно 1000000

Идея для миссии позаимствована с соревнований ICPC
